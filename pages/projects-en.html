🔑 Password Generator
TOOL
▶

Tool to generate configurable secure passwords. Calculates entropy and strength level in real time based on length and complexity using the Web Crypto API.

JavaScript Crypto API Entropy Calc
Stack — Technologies
  • Frontend Web: Semantic HTML5, Vanilla CSS3 (Dark/Minimalist style), ES6+ JavaScript (Mobile-First).
  • Desktop (GUI): Python 3 with standard libraries tkinter and tkinter.ttk.
  • Workflow & Deploy: VS Code, Git/GitHub, GitHub Pages for static web hosting.
🛡️ Security
  • Web Crypto API: Exclusive use of window.crypto.getRandomValues() to ensure maximum entropy with a robust CSPRNG resistant to algorithmic prediction.
  • Strong Cryptography (Desktop): Implementation of Python's native secrets module, discarding the insecure use of random.
  • Zero-Knowledge Local: All generation happens 100% locally, with no server requests, cookies or database, reducing exposure to Man-in-the-Middle interception.
