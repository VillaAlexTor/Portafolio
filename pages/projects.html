🔑 Generador de Contraseñas
HERRAMIENTA · SECURITY
▶

Herramienta para generar contraseñas seguras configurables. Calcula entropía y nivel de fortaleza en tiempo real según longitud y complejidad usando la Web Crypto API.

JavaScript Crypto API Entropy Calc
Stack — Tecnologías
  • Frontend Web: HTML5 Semántico, CSS3 Vanilla (Estilo Dark/Minimalista), JavaScript ES6+ (Mobile-First).
  • Desktop (GUI): Python 3 con librerías estándar tkinter y tkinter.ttk.
  • Workflow & Deploy: VS Code, Git/GitHub, GitHub Pages para hosting web estático.
🛡️ Seguridad
  • Web Crypto API: Uso exclusivo de window.crypto.getRandomValues() para asegurar máxima entropía con un CSPRNG robusto frente a predicción algorítmica.
  • Criptografía Fuerte (Desktop): Implementación del módulo nativo secrets de Python, descartando el uso inseguro de random.
  • Zero-Knowledge Local: Toda la generación ocurre 100% en local, sin peticiones a servidores, cookies ni base de datos, reduciendo exposición a intercepción tipo Man-in-the-Middle.
